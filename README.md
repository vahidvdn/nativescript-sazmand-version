# NativeScript App Version (nativescript-sazmand-version)

## Installation


```
tns plugin add nativescript-sazmand-version
```

## Usage


```JS	
    import { SazmandVersion } from 'nativescript-sazmand-version';
    private appVersion: SazmandVersion;

    constructor() {
	    this.appVersion.get()
  	}
    
```
